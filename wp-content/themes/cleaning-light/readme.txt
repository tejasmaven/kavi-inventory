== Description ==

Cleaning Light is a fast and clean multi-purpose free WordPress theme designed for a wide variety of websites. You can use this theme to launch a cleaning, showcase services, and enhance your website—all without any coding knowledge. It is perfect for cleaning such as construction, restaurants, churches, education, consulting, fitness, roofing, medical, spa, studios, sports, online agencies, charities, and freelancers. The theme includes advanced customizer sections like slider, about us, features, services, call to action, testimonials, team, contact, client logo, portfolio, how it works, video, counter, and more. Each section offers extensive customization options, including background and font colors, gradient colors, margins, padding, and top/bottom separators using SVG graphics. These features make it easy for users to create visually appealing and fully personalized web pages. Cleaning Light also provides 25+ pattern blocks and 40+ Elementor blocks, giving users even more design flexibility. It is suitable for both beginners and experienced users. Additionally, the theme is responsive, Schema.org compatible, browser compatible, SEO-friendly, speed optimized, RTL compatible, and translation ready—ensuring your website is accessible, user-friendly, and search engine optimized so your audience can easily find you online. If you encounter any issues while using our theme, you can refer to our theme documentation or reach out to our friendly support staff at https://ikreatethemes.com/support/.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
